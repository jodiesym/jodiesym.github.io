Hello!

The purpose of this project is to analyze how weather changes as you get closer to the equator. To finish this project we built an API connection to the OpenWeatherMap website that pulled weather data from over 500 cities worldwide.
This project has compared latitude with four variables: temperature, humidity, cloudiness, and wind speed. This site provides the visualized plots of these four variables and also the source data. Please feel free to use the navigation bar to explore all the data and plots we have and learn about it!

Summary : Our study turned out that there is a strong relationshp between latitude and temperature. When latitude is closer to the equator, the temperature will be higher too. For latitude vs. wind speed, humidity, and cloudiness, there is no strong relationshp between them. Even if there was, these would be weak correlations, and we may need a large dataset to prove it.
